# Contexte du projet
1. # Contexte

Le secteur de la navigation de plaisance utilise majoritairement des systèmes de pilote automatiques commerciaux. Ces systèmes sont vendus à des prix coûteux ( 6200 euros en moyenne). Il existe des solutions open-source permettant de faire fonctionner séparément les différents éléments composant un système de pilote automatique. Ces solutions sont moins coûteuses qu’acheter un système commercial mais nécessite une certaine expertise technique.

Notre documentation s’inscrit dans ce contexte. Notre objectif est de proposer un guide reproductible permettant à tout utilisateur de pouvoir construire un pilote automatique de bateau fonctionnel. L’objectif principal est que tout utilisateur non-compétent techniquement puisse parvenir à utiliser ce document. 

Le système de pilote automatique réalisé en suivant cette documentation aura un prix final minimum de 574,2 euros.

Ce système comprend 3 blocs :
Une interface (IHM) permettant de visualiser en temps réel le cap et la position du bateau 
Un anémomètre permettant de mesurer le sens et la vitesse du vent
Un pilote automatique

Cette documentation a été réalisée dans l’objectif d’être un contenu open-source accessible à tous. Elle utilise, de ce fait, uniquement du contenu libre de droits : https://github.com/AndreasW29.
