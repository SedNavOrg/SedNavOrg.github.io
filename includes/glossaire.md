*[Connecteurs de bornier]: Connecteur vert qui permet de connecter les câbles aux cartes électroniques.
*[Hardware]: Éléments matériels d’un système (ex : un câble, une résistance …)
*[IHM]: Interface Homme Machine, partie qui fait le lien entre l’utilisateur et le système pour faciliter l’utilisation des différents outils.
*[OpenPlotter]: Ensemble logiciel et matériel ayant pour but d’aider à la navigation sur les voiliers de petite et moyenne taille
*[OS]: Operating System, quelques exemples sont Windows, MacOS ou encore linux
*[PID]: Proportionnel, Intégral, Dérivateur (lien suivant si intéressés : [https://fr.wikipedia.org/wiki/R%C3%A9gulateur\_PID](https://fr.wikipedia.org/wiki/R%C3%A9gulateur_PID)) 
*[Software]: Éléments immatériels d’un système (ex : logiciels)
*[ASA]: L'évolution de l'ABS, un polymère utilisé nottament pour l'impression 3D
